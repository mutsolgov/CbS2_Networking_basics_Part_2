    R1#configure terminal
1.1. Настрой DNS-службу:
    R1(config)#ip dns server

1.2. Добавь запись DNS для домена my.site, который будет указывать на loopback-интерфейс:
    R1(config)#interface loopback0
    R1(config-if)#ip address 1.1.1.1 255.255.255.255
    R1(config-if)#exit

    R1(config)#ip host my.site 1.1.1.1
    Это добавляет запись "my.site → 1.1.1.1" в DNS-сервер.



Шаг 2. Настройка клиента DNS (например, на R2)
2.1. Укажи DNS-сервером IP-адрес R1:
    R2#configure terminal
    R2(config)#ip name-server 10.0.0.1
    (Предполагаем, что между R1 и R2 настроена связь и R1 доступен по адресу 10.0.0.1)

    R2(config)#ip route 1.1.1.1 255.255.255.255 10.0.0.1


    ✅ 4. Включи DNS lookup на R2 (если был отключен)
    R2(config)#ip domain-lookup

3.2. Проверь DNS-разрешение:
    R2#ping my.site
